## CAP-394
Repositório para disciplina Introdução a Data Science (CAP-394), do Instituto Nacional de Pesquisas Espaciais.

## Instalação


$ conda create -n cap394 python=3.6
$ conda activate cap394
$ conda install jupyter
$ jupyter notebook

Packages
- numpy=1.17.0
- matplotlib.pyplot
- xarray
- arm_pyart
- basemap
- basemap-data-hires
